# 🚚 Gestor de Entregas v3.0

Sistema de gestión de pedidos y entregas con notificaciones en tiempo real.

## 🚀 Características

- 👥 **Gestión de Usuarios**: Crear, activar/desactivar, eliminar repartidores
- 📦 **Gestión de Pedidos**: Crear, asignar, completar pedidos
- 💰 **Liquidaciones**: Cálculo automático de liquidaciones para repartidores
- 🔔 **Notificaciones Push**: Alertas en tiempo real cuando hay nuevos pedidos
- 🔊 **Sonido**: Notificación sonora al recibir un nuevo pedido
- 📱 **Responsive**: Funciona en PC, tablet y móvil

## 🔧 Tecnologías

- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: Firebase Realtime Database
- **Notificaciones**: Push API + Web Audio API
